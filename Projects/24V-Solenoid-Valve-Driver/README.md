# 24V Solenoid Valve Driver

## Project Overview

This project was developed to better understand actuator control systems commonly used in irrigation products. The design demonstrates the electrical hardware required to safely control a 24V solenoid valve using a low-side MOSFET switching architecture.

### Key Features

- 24V Solenoid Control
- AO3400 Logic-Level MOSFET
- Flyback Diode Protection
- Reverse Polarity Protection
- 3.3V / 5V MCU Compatibility
- Custom PCB Design in Altium Designer

## Design Requirements

| Parameter | Value |
|------------|------------|
| Supply Voltage | 24VDC |
| Load Type | Solenoid Valve |
| Control Voltage | 3.3V / 5V Logic |
| Switching Method | Low-Side MOSFET |
| Flyback Protection | Yes |
| Reverse Polarity Protection | Yes |

## Design Approach

A low-side switching architecture was selected due to its simplicity and widespread use for inductive load control.

The solenoid remains connected to the 24V supply while the MOSFET controls the return path to ground. When the MOSFET is enabled, current flows through the solenoid and actuates the valve.

A flyback diode was included across the load to suppress inductive voltage spikes generated when the solenoid is de-energized.

## Major Components

### AO3400 MOSFET

Selected due to:

- Logic-level gate drive
- Low RDS(on)
- Compact footprint
- Suitable current handling capability

### Flyback Diode

Provides protection against inductive kickback generated by the solenoid coil.

### Reverse Polarity Protection

Added at the input connector to prevent accidental damage from incorrect wiring.

## PCB Layout Considerations

- Wide traces used for power routing
- Ground pour implemented for low impedance return paths
- Flyback diode placed near load connection
- MOSFET positioned close to current path
- Clearly labeled connectors for field wiring
