# Flexible BLE Pressure Sensor

## Overview

Designed a custom modular PCB system for wireless pressure sensing and real-time BLE transmission for dysphagia research.

## Key Technologies

- nRF52840 BLE SoC
- STM32 Microcontroller
- 2-Layer Flexible PCB Design
- Force Sensitive Resistors (FSRs)
- Signal Conditioning
- Bluetooth Low Energy

## Engineering Skills Demonstrated

- PCB Design
- Component Selection
- Hardware Validation
- Embedded Systems
- BLE Communications
- Sensor Integration
